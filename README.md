Página web responsiva, moderna e informativa para un negocio local de abarrotes, desarrollada como preparación para su despliegue en un servidor web Linux dentro de un entorno VPS.

## Objetivo
Diseñar e implementar una página web informativa utilizando HTML5, CSS3, JavaScript y el framework Bootstrap 5, aplicando diseño responsivo, estructura modular y buenas prácticas de desarrollo front-end, lista para ser desplegada en un servidor VPS con Linux.

## Descripción del Proyecto
El sitio web corresponde a la Tienda de Abarrotes Doña Mari, un negocio local con más de 25 años de experiencia. La página presenta:

Nombre del negocio y descripción general.

Historia y valores (sección "Nosotros").

Servicios ofrecidos (productos frescos y pagos flexibles).

Catálogo visual mediante un carrusel de imágenes.

Promociones dinámicas.

Información de contacto, horarios y ubicación.

Formulario de contacto y suscripción a newsletter.

El diseño sigue una línea de branding basada en colores verde y naranja, que evocan frescura, tradición y calidez.

## Tecnologías Utilizadas
Tecnología	Descripción
HTML5	Estructura semántica del documento.
CSS3	Estilos personalizados para complementar Bootstrap.
Bootstrap 5	Framework CSS para diseño responsivo y componentes (navbar, cards, grid, carrusel, botones).
JavaScript	Interactividad: horarios dinámicos, cambio de promociones, validación de formularios, navegación suave.
Bootstrap Icons	Íconos vectoriales para mejorar la interfaz.
Google Fonts / System Fonts	Tipografía legible y moderna.
## Estructura del Proyecto
El proyecto consta de un único archivo HTML que incluye todo el código necesario (CSS embebido, JavaScript embebido), lo que facilita su despliegue en entornos simples.

## Características Principales
Diseño 100% Responsivo, 
se adapta a móviles, tablets y escritorio gracias al sistema de grid de Bootstrap y media queries personalizadas.

Navbar colapsable en dispositivos pequeños.

## Componentes Bootstrap
Navbar fijo con enlaces de navegación suave.

Grid system para organizar contenido.

Cards para presentar servicios.

Carrusel con imágenes representativas de los productos.

Botones estilizados con colores de la marca.

## Branding Personalizado (CSS)
Colores institucionales: #2e7d32 (verde) y #ff9800 (naranja).

Efectos hover en tarjetas y enlaces.

Fondo hero con overlay oscuro y tipografía prominente.

Estilos exclusivos para horarios, promociones y footer.
## Interactividad con JavaScript
Horario dinámico: Muestra el horario del día actual y permite alternar a la tabla completa.

Carrusel automático: Controlado por Bootstrap JS, con imágenes de productos.

Promoción del día: Botón que cambia aleatoriamente entre diferentes ofertas.

Formulario de contacto: Simula envío de mensaje con alerta de confirmación.

Suscripción a newsletter: Validación básica de email y mensaje de agradecimiento.

Navegación suave: Scroll animado al hacer clic en los enlaces del navbar.

Resaltado de sección activa: La opción del menú se actualiza según el scroll.

## Clonar y Ejecutar Localmente
Puedes clonar este repositorio y visualizar la página web directamente en tu navegador sin necesidad de un servidor web complejo.

## Prerrequisitos
Git instalado en tu sistema.

Un navegador web moderno (Chrome, Firefox, Edge, etc.).

## Pasos para clonar y ejecutar
Clonar el repositorio

bash
git clone https://github.com/victordanielsainz07-create/Abarrotes_Mari.git
Reemplaza la URL con la dirección real del repositorio.

Acceder a la carpeta del proyecto

cd tienda-dona-mari
Abrir el archivo HTML

Opción A: Haz doble clic en el archivo index.html desde tu explorador de archivos.

Opción B: Abre tu navegador y arrastra el archivo index.html a la ventana.

Opción C: Desde la terminal, abre con el navegador por defecto:
# En Linux/macOS
open index.html

# En Windows (PowerShell)
start index.html
 La página se cargará completamente, con todas las funcionalidades interactivas funcionando.
